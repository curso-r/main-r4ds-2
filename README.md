## Plano de aula

6 aulas (18h) com os temas:

- Manipulação de dados avançada
- Pacote stringr
- Pacote lubridate, pacote forcats
- Controle de fluxo (`if`, `while` e `for`)
- Pacote purrr
- Non-standard evaluation
